// setup guide
01
Create your Telegram Bot
Open Telegram and message @BotFather. Send /newbot, give it a name, and copy the bot token you receive.
02
Get your Telegram User ID
Message @userinfobot on Telegram — it will reply with your numeric user ID. Save this.
03
Get your Anthropic API Key
Visit console.anthropic.com, create an account if needed, and generate an API key. The bot uses Claude Sonnet with web search.
04
Install & configure
# Clone / create your project folder
mkdir airdrop-radar && cd airdrop-radar

# Install dependencies
pip install -r requirements.txt

# Set your environment variables
export TELEGRAM_BOT_TOKEN="your_token_here"
export TELEGRAM_USER_ID="your_user_id_here"
export ANTHROPIC_API_KEY="your_api_key_here"

# Run the bot
python bot.py
05
Keep it running 24/7 (optional but recommended)
Use a cheap VPS (Hetzner CX11 ~€4/mo, or a free Oracle Cloud instance). Run with pm2 start bot.py --interpreter python3 or use systemd to keep it alive permanently. The 8AM digest uses your server's timezone — set it with timedatectl set-timezone Africa/Lagos (or your zone).
